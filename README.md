# JavaScript Projects

This repository contains a collection of small projects built using **HTML, CSS, and JavaScript**. These projects were created to practice core JavaScript concepts, logic building, and DOM manipulation.

## Projects Included

### Pizza Menu

An interactive pizza ordering menu that allows users to choose from multiple sizes and toppings. Based on the selections, the application calculates and displays the final price dynamically.

**Key Features:**

* Multiple topping options
* Dynamic price calculation
* User-friendly interface

---

### Tic-Tac-Toe Game

A digital version of the classic Tic-Tac-Toe game. The game randomly determines which player starts first, allows players to choose their symbols, and celebrates the winner with visual and sound effects.

**Key Features:**

* Random first-player selection
* Custom player tokens
* Win animations and sound effects

---

### Calculator

A clean and simple calculator that performs basic arithmetic operations. Users can add, subtract, multiply, and divide using an easy-to-use web interface.

**Key Features:**

* Four basic math operations
* Responsive layout
* Minimal and sleek design

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
